# Auth System

Um sistema de autenticação e gerenciamento de usuários desenvolvido em **Java** usando **Spring Boot**, **Spring Security**, **JPA/Hibernate** e **PostgreSQL**. Este projeto serve como base para aplicações que precisam de registro, login, controle de roles e permissões.

---

## 📝 Funcionalidades

- Registro de usuários
- Login e autenticação com **JWT**
- Controle de roles (`USER`, `ADMIN`, etc.)
- Inicialização de dados básicos (roles padrão)
- Tratamento global de exceções
- Estrutura modular e organizada para fácil manutenção

---

## 💻 Tecnologias

- **Java 8**
- **Spring Boot 2.7**
- **Spring Security**
- **Spring Data JPA / Hibernate**
- **PostgreSQL**
- **Lombok**
- **Maven**
- **JWT (JSON Web Token)**
